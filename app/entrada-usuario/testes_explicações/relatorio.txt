Explicações dos Testes

=== CodigoUsuario_testes.java ===

Este arquivo de teste contém 14 testes para a classe ConversorUnidades. Os testes validam as funções de conversão entre diferentes unidades de medida, como metros, centímetros, quilômetros, litros e mililitros.

Os métodos testados incluem:
- metroParaCentimetro
- centimetroParaMetro
- kilmetroParaMetro
- litroParaMililitro
- main

Os cenários principais incluem:
- Conversão de unidades de medida positivas
- Conversão de unidades de medida negativas
- Conversão de unidades de medida zero
- Conversão de unidades de medida nulas (com exceção de NullPointerException)

O uso de mocks não é necessário, pois os testes estão direcionados para a classe ConversorUnidades.

As exceções testadas incluem:
- NullPointerException para conversão de unidades de medida nulas

Os testes também validam a saída da função main da classe ConversorUnidades.
