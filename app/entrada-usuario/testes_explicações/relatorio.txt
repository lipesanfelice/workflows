Explicações dos Testes

=== CodigoUsuario_testes.java ===

Este arquivo de teste valida a funcionalidade da classe CalculadoraEstatistica, que é responsável por calcular a média e a mediana de um vetor de números.

O teste cobre os métodos calcularMedia e calcularMediana da classe CalculadoraEstatistica.

Os cenários principais incluem:

- Vetor vazio
- Vetor com um elemento
- Vetor com dois elementos
- Vetor com três elementos
- Vetor com quatro elementos
- Vetor com cinco elementos
- Vetor com elementos negativos
- Vetor com elementos iguais
- Vetor com elementos diferentes

O teste utiliza mocks para simular a entrada de dados para os métodos calcularMedia e calcularMediana.

As exceções esperadas incluem:

- Nenhuma exceção é esperada, pois os métodos calcularMedia e calcularMediana são projetados para lidar com vetores vazios e com elementos de diferentes tipos.

O contexto desse teste é a validação da funcionalidade da classe CalculadoraEstatistica, que é uma classe importante para o cálculo de estatísticas em um sistema de gerenciamento de dados.
